# Sales Performance Dashboard (Power BI)

## Overview
This project presents an interactive sales dashboard built using Power BI to analyze business performance.

## Features
- Total Revenue, Orders, and Customer KPIs
- Monthly sales trends analysis
- Product-wise and country-wise performance
- Interactive filters and slicers

## Tools Used
- Power BI
- DAX
- Data Visualization

## Insights
- Identified top-performing products and regions
- Analyzed monthly sales patterns
